# 🚀 Generate Portfolio

**Generate Portfolio** is a web application that helps users quickly create a professional developer portfolio.
Instead of designing a portfolio from scratch, users can simply provide their personal details, skills, and project information to automatically generate a clean and responsive portfolio website.

This project simplifies the process of building a personal website and helps developers showcase their projects, skills, and achievements in an organized and visually appealing way.

---

## ✨ Features

* ⚡ Generate a portfolio instantly
* 👩‍💻 Add personal details (name, bio, contact information)
* 🛠 Showcase technical skills and technologies
* 📂 Display projects with descriptions and links
* 🌐 Responsive design for mobile and desktop
* 🎨 Modern UI for professional presentation
* 🔗 Social media and GitHub integration

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/VidhiPandit19/generate-portfolio.git
```

Move into the project directory:

```bash
cd generate-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
